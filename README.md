# Healthcare Management System using Spring Boot Microservices

## Overview
Backend healthcare management system developed using Java and Spring Boot microservices architecture. The application consists of independent services for patient management, billing, and analytics with scalable and modular backend design.

## Tech Stack
- Java
- Spring Boot
- Microservices
- Apache Kafka
- gRPC
- PostgreSQL
- Docker
- Maven

## Features
- Patient Management Service
- Billing Service
- Analytics Service
- API Gateway
- Event-driven communication using Kafka
- Inter-service communication using gRPC

## Architecture
Client → API Gateway → Microservices → PostgreSQL/Kafka

## Project Structure
- patient-service
- billing-service
- analytics-service
- api-gateway

## GitHub Repository
Backend microservices project demonstrating scalable distributed system architecture.
